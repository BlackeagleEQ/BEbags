# BEbags

BEbags is a MacroQuest Lua script for AscendantEQ that combines all your inventory bags into a single, easy-to-use interface, with optional bank viewing and quality-of-life features.

## Features
- Unified inventory bag view
- Synced bank browser
- Deposit, Destroy, and Drop actions
- Double-click item inspect
- Diablo-style UI + 4 theme presets:
  - Classic
  - Diablo
  - Emerald
  - Frost
- Quick launcher and config menu

## Installation
1. Download and extract this package
2. Place `BEbags.lua` into your MacroQuest `/lua` folder
3. In game, run:
   `/lua run BEbags`

## Commands
- `/BEbags` - Toggle main window
- `/BEbags config` - Open config
- `/BEbags help` - Open help
- `/BEbags destroy` - Destroy item on cursor
- `/BEbags drop` - Drop item on ground

## Notes
- Designed for AscendantEQ
- Bank actions require bank window open for live interaction
- Destroy/Drop actions are immediate (no confirmation)
